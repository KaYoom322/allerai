# AllerAI (Allergy Scanner for Travelers)

> 식품 알레르기가 있는 외국인을 위한 한국 가공식품 알레르기 성분 감지 및 대체 식품 추천 웹 서비스

## 프로젝트 소개 (Project Overview)
AllerAI는 한국의 가공식품 성분표를 읽기 어려운 외국인들을 위한 서비스입니다. 
스마트폰 카메라로 가공식품 뒷면의 상세 영양 정보를 촬영(OCR)하면, 사용자가 미리 등록한 알레르기 유발 성분과 대조하여 안전 여부를 영어로 안내합니다. 알레르기 유발 물질이 포함된 경우, 공공데이터 포털의 해썹(HACCP) API를 활용하여 동일한 식품 유형 내에서 안전하게 섭취 가능한 대체 식품을 추천합니다.

---

## 주요 기능 (Key Features)

| 기능 분류 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| 회원가입 및 로그인 | ID(이메일), 비밀번호, 전화번호, 알레르기 정보 입력 및 관리 | 사용자 맞춤형 설정 |
| OCR 성분 인식 | 카메라 촬영을 통해 한국 가공식품 뒷면의 텍스트 및 식품 유형 추출 | 실시간 이미지 분석 |
| 알레르기 위험 감지 | 매칭되는 알레르기 성분 발견 시 영어 경고, 안전할 시 안전 메시지 출력 | 글로벌 편의성 제공 |
| HACCP 대체 식품 추천 | 공공데이터 포털 HACCP API 연동을 통해 동일 식품 유형 내 알레르기 프리 제품 추천 | 안심 대안 알고리즘 |
| 빠른 재촬영 | 결과 확인 페이지에서 버튼 클릭 한 번으로 카메라 모드로 즉시 복귀 | 사용자 경험(UX) 고려 |

## 서비스 흐름도 (Service Flow)

1. WELCOME (로그인 / 회원가입)
   - 앱 타이틀 표시 및 가입/로그인 진행
   - 회원가입 시 알레르기 유발 성분은 객관식 선택형으로 등록

2. 홈 화면 (My Home)
   - 저장된 사용자 정보(전화번호, 알레르기 성분) 확인 및 카메라 이동 버튼 제공

3. 카메라 촬영 및 OCR 분석
   - 가공식품 뒷면 촬영을 통한 텍스트 및 식품 유형 식별

4. 결과 안내 및 추천 (Result & Recommendation)
   - 알레르기 포함 여부 결과 및 안전 식품 추천 리스트 제공
   - 다시 촬영하기 버튼을 통한 순환 구조
